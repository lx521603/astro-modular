**Google 公共 DNS IP 地址**

Google 公共 DNS IP 地址 (IPv4) 如下所示：

```
8.8.8.8
8.8.4.4
```
Google 公共 DNS IPv6 地址如下所示：

```
2001:4860:4860::8888
2001:4860:4860::8844
```
DNS-over-TLS
```
dns.google
```
DNS-over-HTTPS (DoH)
```
https://dns.google/dns-query - RFC 8484（GET 和 POST）
https://dns.google/resolve？- JSON API (GET)
```
